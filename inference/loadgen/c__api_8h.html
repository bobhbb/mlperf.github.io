Detailed Description

A C API wrapping the C++ loadgen. Not tested. Needs work.

The C API allows a C or Python client to easily create a SystemUnderTest without having to expose the SystemUnderTest class directly. ConstructSUT works with a bunch of function poitners instead that are called from an underlying trampoline class.
