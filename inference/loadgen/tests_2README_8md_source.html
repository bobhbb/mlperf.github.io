1 # Building and Running the Tests {#ReadmeTests}
2 
3 The unit and performance tests are only supported via gn/ninja at the moment.
4 
5 See the [top-level build readme](@ref ReadmeBuild) for details but, from a clean checkout, you must first run:
6 
7  make bootstrap_gn_ninja
8  third_party/gn/gn gen out/Release --args="is_debug=false"
9 
10 This will build the gn and ninja build tools and create a release project.
11 
12 ## Unit Tests
13 
14 To build:
15 
16  third_party/ninja/ninja -C out/Release mlperf_loadgen_tests_basic
17 
18 To run all tests:
19 
20  out/Release/mlperf_loadgen_tests_basic .
21 
22 To run specific tests:
23 
24  out/Release/mlperf_loadgen_tests_basic <regex>
25  e.g.:
26  out/Release/mlperf_loadgen_tests_basic SingleStream
27 
28 ## Performance Tests
29 
30 To build:
31 
32  third_party/ninja/ninja -C out/Release mlperf_loadgen_perftests
33 
34 To run all tests:
35 
36  out/Release/mlperf_loadgen_perftests .
37 
38 To run specific tests:
39 
40  out/Release/mlperf_loadgen_perftests <regex>
41  e.g.:
