35 /// \brief SUT calls this to notify loadgen of completed samples.
36 /// \details
37 /// * The samples may be from any combination of queries or partial queries as
38 /// issued by \link mlperf::SystemUnderTest::IssueQuery
39 /// SystemUnderTest::IssueQuery \endlink.
40 /// * The SUT is responsible for allocating and owning the response data
41 /// which must remain valid for the duration of this call. The loadgen
42 /// will copy the response data if needed (e.g. for accuracy mode).
43 /// + Note: This setup requires the allocation to be timed, which
44 /// will benefit SUTs that efficiently recycle response memory.
45 /// * All calls to QuerySampleComplete are thread-safe and wait-free bounded.
46 /// + Any number of threads can call QuerySampleComplete simultaneously.
47 /// + Regardless of where any other thread stalls, the current thread will
48 /// finish QuerySampleComplete in a bounded number of cycles.
